# Algoritmos y Estructuras de Datos

Este repositorio contiene mis implementaciones de algoritmos clásicos de ordenamiento, recursión y estructuras de datos desarrollados principalmente en lenguaje C, C++, Java como material de referencia desarrollados durante mis estudios de ingeniería en sistemas.

## Estructuras de Datos
Implementaciones de estructuas de datos en C manejo de memoria dinámica. 
* Lista enlazada
* Lista doblemente enlazada
* Cola estática, dinámica y circular
* Automata de Pila
* Árboles (Binario de búsqueda, AVL)
* Gráfos
