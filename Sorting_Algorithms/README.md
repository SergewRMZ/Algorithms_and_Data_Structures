# Algoritmos de Ordenamiento

Codificación de 10 de los algoritmos de ordenamiento más conocidos en las ciencias de la computación.

# Ejecución de los programas
Cada programa se ejecuta de manera individual, al ejecutar el programa recibe como parámetro la cantidad de números por ordenar y podemos mandarle un archivo con los números que queremos ordenar de la siguiente manera:

```
    ./nombrePrograma 100 > archivoSalida.txt
```

# Mandar datos de un archivo

```
    ./nombrePrograma 100 < datos.txt > archivoSalida.txt
```
